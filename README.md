# AnimeLek Scraper

Simple web scraper for [AnimeLek](https://animelek.top/).

- Search anime by name
- Browse latest episodes

## Usage

1. Enter an anime name in English or Arabic
2. Click **Search**
3. Or click **Latest Episodes** to browse pinned episodes
